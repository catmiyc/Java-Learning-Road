## 类和对象
万物皆可对象，客观存在的事物皆为对象

类： 对现实生活中一类具有共同属性和行为的食物的抽象

类的组成：<font color = "red">属性</font>和<font color = "red">行为</font>
- 属性：在类中通过成员变量来体现
- 行为：在类中通过成员方法来体现

```java
public class类名{
  //成员变量

  //成员方法
}
```

### 对象的使用
1.创建对象：
格式：
```java
类名 对象名 = new 类名();
```

范例：
```java
Phone p = new Phone();
```

2.使用对象：

对象使用成员变量：
格式：
```java
对象名.变量名
```
范例：
```java
p.brand
```

对象使用成员方法：
格式：
```java
对象名.方法名();
```
范例：
```java
p.call();
```
String s 是要大写的
